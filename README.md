# Shutdown Manager Pro

## 🚀 Kurulum & Çalıştırma

Clone the repository:

```bash
 git clone https://github.com/isaphlivan/Shutdown-Manager-Pro.git
```

Navigate into the directory:

```bash
 cd Shutdown-Manager-Pro
```

## ⚙️ System Requirements

- Windows 10/11 (x64)
- macOS High Sierra or later
- Administrator/sudo privileges for system commands
- Node.js v14+

## Troubleshooting

If you encounter any issues during installation or usage, please check the following:
- Ensure you have the correct permissions to run the commands.
- Check Node.js version with `node -v` command.
- Verify the repository is cloned correctly and all dependencies are installed.

For further assistance, please open an issue on GitHub.
